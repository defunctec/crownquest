Welcome to the CrownQuest wiki!

Here, you can learn all about CrownQuest, the first Minecraft server with a Crown-denominated economy. Please remember that this server is still under development, so check this wiki frequently to stay up-to-date with changing mechanics.

To get started, you can take a look at the README.md page. For more information about CrownQuest's community, mechanics, and development, read the following:

The CrownQuest Community

CrownQuest started as an experiment of what creating a game with an economy based off a real currency could do. Of course, at the center of any economy is the community, and CrownQuest has created an interesting one.

The easiest way to get involved is to join CrownQuest's server via Minecraft 1.13.2 with the address, play.crownquest.co (for Crown server) or play.crownquest.co (for Dogecoin)

You can also join CrownQuest's Discord server to get the latest community news and talk to other CrownQuest players.

CrownQuest doesn't filter its chat, but in case of proof of abusive chat, some users should be banned.
